# @xai-official/intelekt-linux-arm64

Platform-specific binary for [`@xai-official/intelekt`](https://www.npmjs.com/package/@xai-official/intelekt) on linux-arm64.

Do not install this package directly. Install the main package instead:

```sh
npm install -g @xai-official/intelekt
```

The main package will automatically pull the correct binary for your platform via `optionalDependencies`.
